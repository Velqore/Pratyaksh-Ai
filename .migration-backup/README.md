## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Modern web browser

### Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/pratyaksh-forensic-ai.git
cd pratyaksh-forensic-ai

# Install dependencies
npm install

# Start development server
npm run dev

# Access the application
open http://localhost:8080
```

### Production Build
```bash
# Build for production
npm run build

# Start production server
npm start

# Test the build
curl http://localhost:8080/api/ping
```

## 🏗️ Architecture

### Frontend (React SPA)
- **Framework**: React 18 + TypeScript
- **Styling**: TailwindCSS 3 + Custom Forensic Theme
- **UI Components**: Radix UI + Custom Components
- **State Management**: React Query + useState
- **Routing**: React Router 6 (SPA mode)

### Backend (Express Server)
- **Runtime**: Node.js + Express
- **API Design**: RESTful endpoints
- **Development**: Hot reload integration
- **Production**: Optimized builds

### AI Response System
- **Intelligence**: Realistic forensic AI responses
- **Departments**: Specialized knowledge domains
- **Accuracy**: Simulated 98%+ confidence scores
- **Analysis**: Multi-step forensic workflows
